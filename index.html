Automatización con Jenkins & Kubernetes

Pipeline CI/CD completo con despliegue en K3s

¿Qué hace esta solución?

Este proyecto integra Jenkins con Kubernetes (K3s) para automatizar el despliegue continuo de una aplicación. Desde el push al repositorio hasta la ejecución en producción, todo está orquestado.

Arquitectura del Pipeline

  • GitHub: Gestión del código fuente.
  • Jenkins: Construcción de la imagen Docker y despliegue.
  • Docker: Construcción de contenedores.
  • Docker: desarrollo en nextJS.
  • K3s: Cluster Kubernetes ligero donde se ejecuta la app.

Ventajas

Automatización total

Desde el push hasta el despliegue, sin intervención manual.

Escalabilidad

Gracias a Kubernetes, tu app escala fácilmente.

Control de versiones

Cada commit puede generar una imagen trazable y reproducible.

Despliegue seguro

Entorno aislado y controlado con namespaces y RBAC.
